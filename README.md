# ReactIntervalPiano
A Javascript piano app whose keys plays intervals rather than fixed notes.

### Prerequisites

Node.js

### Installing

npm build
npm install

## Built With

* [React.js](https://reactjs.org/) - Javascript Framework
* [Pizzicatoo.js](https://alemangui.github.io/pizzicato/) - Javascript Web Audio Library
